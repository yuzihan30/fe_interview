1. 通过Promise实现每隔1秒输出1、2、3
```
使用组合函数, 上一个函数的返回值作为下一个函数的入参
[1, 2, 3].reduce((p, x) => {
    return p.then(() => {
        return new Promise(r => {
            setTimeout(() => r(console.log(x)), 1000)
        })
    })
}, Promise.resolve())

简洁写法：
[1, 2, 3].reduce((p, x) => p.then(() => new Promise(r => setTimeout(() => r(console.log(x)), 1000))), Promise.resolve())
```
